# Hibernate-MySQL-Project

A program for database management (**MySQL**) and perform sample **CRUD** operations from within Java code using Hibernate (**ORM**).

### Types of created relationships

- @OneToOne
- @OneToMany/@ManyToOne
- @OneToMany One-way
- @ManyToMany

### Technologies

Project is created with:

- Hibernate 5.3.6
- MySQL 8.0
- Maven
- HQL
